# 4Bottels

En simpel React Native app til at holde styr på at drikke 2 liter vand om dagen.

## Funktioner
- Viser en vandflaske, der starter på 2000 ml.
- Indtast hvor meget du har drukket, og flasken tømmes tilsvarende.
- Daglig reset til fuld flaske.
- Tæller hvor mange dage i træk du har drukket hele flasken.

## Kom i gang
```bash
npm install
npm run ios
# eller
npm run android
```
